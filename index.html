Introduction

All programming languages have their own built-in data structures, often differing from other languages.

JavaScript has seven primitive values: String, Number, BigInt, Boolean, Undefined, Null and Symbol.

Everything else is an Object. Common Object types include Functions and Arrays.

String

The String type represents textual data through UTF-16 code units. The length of the string is the number of UTF-16 code units in it. Each element in the string occupies a position, the first element is at index 0, the second at index 1, and so on. Strings are immutable values and cannot be modified once created.

  • Double quotes and single quotes are interchangable.
  • Back ticks are used when you need to include variables in the string.
let pet1 = "cat";
let pet2 = 'dog';
let result = `Alex has a ${pet1} and Jo has a ${pet2};
//Alex has a cat and Jo has a dog
Number

Simply put, the Number type stores positive and negative floating-point numbers. There is however a specific range that can be safely stored.

let x1 = 31;
let x2 = -12.5;

The BigInt type can represent numbers with arbitrary magnitude. With BigInts you can safely store numbers beyond the safe limit for the Number type.

Boolean

The Boolean type will represents one of two values: true or false. They are usually used for conditional operations, for example:

  • ternary operators
  • if...else
  • while
let x = 5;
let y = 7;
let z = 5 (x == y) //returns false
(x == z) //returns true
Undefined

The undefined type is inhabited by only one value: undefined. It indicates the absence of a value.

Null

The null type is inhabited by only one value: null. It indicates the absence of an object.

const number = null
Symbol

The Symbol type is a unique and immutable value. Symbols are often used in Objects to create unique keys that are guaranteed not to clash with keys from other code.

/*hello world*/
Object

In JavaScript, Objects can be seen as a collection of key-value pairs that can be added or removed. Keys can be strings or symbols, and values can be any data type, including other objects.
